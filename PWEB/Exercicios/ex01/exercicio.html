Definição

Um framework é uma estrutura abstrata que fornece suporte genérico para um software. Ele incluibibliotecas, convenções e padrões que facilitam o desenvolvimento de aplicações

um framework é uma arquitetura desenvolvida com o objetivo de atingir a maxima reutilização... Matsson(2000)

Aplicação

Existem frameworks para várias áreas de programação, como:

  1. desenvolvimento web
  2. inteligencia artificial
  3. aplicativos móveis
  4. outros

Exemplos

Alguns exemplos populares de frameworks inclui incluem:
